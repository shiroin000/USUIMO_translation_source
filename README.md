# USUIMO Translation Source

This repository contains the translation source files for **A Simple Life with My Unobtrusive Sister**.

The text in this repository is updated alongside the latest in-game text, so translators can work with the current source material more easily.

## What is included

This repository contains language files used for translation work, including folders such as:

- `data/CN`
- `data/EN`
- `data/JP`
- `data/VN`
- `data/RU`

and other supported language folders when available.

## Purpose

This repository is intended to support community translation collaboration.

Contributors can:

- improve existing translations
- fix grammar, wording, or terminology issues
- help maintain consistency across files
- contribute translations for additional languages

## Important rules

Please follow these rules when editing files:

- Do not modify JSON keys
- Do not change the original file structure
- Do not remove placeholders, escape characters, or control codes
- Keep file encoding as UTF-8
- Try to keep one language per pull request

## How to contribute

1. Fork this repository
2. Create a new branch in your fork
3. Edit only the target language files
4. Commit your changes
5. Open a pull request to this repository

## Pull request notes

When submitting a pull request, please describe:

- which language you edited
- which files you changed
- whether the changes are full translations, corrections, or proofreading

## Validation

Pull requests may be checked automatically for:

- invalid JSON format
- broken structure
- missing required data

If validation fails, please fix the errors and update your pull request.

## Notes

This repository is for translation collaboration.

It does not contain the full game project or executable files.
